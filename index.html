Grid:Css Grid Layout, is a two-dimensional grid-based layout system with rows and columns, making it easier to design web pages without having to use floats and positioning.

Flexbox: The CSS Flexbox offers a one-dimensional layout. It is helpful in allocating and aligning the space among items in a container.It works with all kinds of display devices and screen sizes.

Tailwind: Tailwind CSS uses a set of utility classes to create a neat UI with more felxibility and uniqueness.
Websites created using Tailwind CSS are much more customizable.

Bootstrap: Bootstrap comes with a set of pre-styled responsive, mobile-first componenets that posssesss a definite UI kit.
Sites created using Bootstrap follow the generic pattern that makes them look identical.

Css box model is a container that contains multiple properties including borders, margin, padding, and the content itself. It is used to create the design and layout of web pages. It can be used as a toolkit for customizing the layout of different elements. The web browser renders every element as a rectangular box according to the CSS box model.

Elements such as <header> <footer> <article> are all considered semantic because they accurately describe the purpose of the element and the type of content that is inside them.
